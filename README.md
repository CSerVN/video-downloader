# 🎯 VIDEO DOWNLOADER
---
A powerful desktop application designed to help you download videos and playlists from various platforms effortlessly.

📖 **[Installation Guide](https://github.com/CSerVN/video-downloader/blob/main/Installation.md)**

## ✨ Key Features
There are two main ways to hunt your videos:

- **📋 Smart Clipboard Monitoring:** The app continuously monitors your clipboard. Just copy a video/playlist link, and it will be captured and downloaded automatically. You can also manually input links if you prefer.
- **🕵️ Browser Extension "Hunting" Mode:** Catch media streams directly from your browser! By following the instructions to import our custom extension, it will automatically activate and hunt for videos whenever you browse using the dedicated Chrome profile we generate for you.
